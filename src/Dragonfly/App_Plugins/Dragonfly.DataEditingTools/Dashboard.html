Re-Save All Media Nodes
Sometimes it's necessary to re-save all the media in order to recalculate properties such as file size, etc. Doing this manually can take hours for a large media library. These options will handle it automatically.

NOTE: If a Media node is missing a file, an error will be logged and the process will continue. If you don't want to log missing files, use the second option.
